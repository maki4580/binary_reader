# BinaryReader

## 概要

BinaryReaderは、バイナリファイルから指定された数のデータペアを読み込み、表示するシンプルなCプログラムです。各データペアは2つの16ビット（2バイト）符号なし整数値で構成されています。

## 機能

- バイナリファイルからデータペアの読み込み
- 読み込んだデータを10進数と16進数形式で表示
- 指定した数のデータペアのみを読み込む機能

## 構造

プロジェクトは次のファイルで構成されています：

- **binary_reader.h**: データ型定義とプログラムの関数宣言を含むヘッダーファイル
- **binary_reader.c**: バイナリファイルの読み込みと表示機能を実装したソースファイル
- **main.c**: メインプログラム（コマンドライン引数の処理と全体のフロー制御）
